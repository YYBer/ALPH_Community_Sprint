# ALPH_Community_Sprint

### Backend:
```
cd backend
npm start
```
### or run it in the docker
```
docker build -t competition-recorder-backend .
docker run --env-file .env -d competition-recorder
```
stop the docker
```
docker ps
docker stop <container_id>
docker rm <container_id>
```
